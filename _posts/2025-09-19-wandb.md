### 1. 환경에 wandb 설치
```
pip install wandb 
```

### 2. wandb 로그인
```
wandb login
```

### 3. wandb 시작
```
wandb init
```

### 4. wandb 적용
- wandb를 적용하고 싶은 training 혹은 test 파일에 wandb를 import
- wandb 로그를 기록하고 싶다면, wandb.log 설정
```
import wandb
wandb.init(project="생성한 project 이름", config={
    "learning_rate": 0.0001,
    "dropout": 0.2,
    "architecture": "CNN",
    "dataset": "COCO",
})
config = wandb.config
wandb.log({"acc":acc, "loss":loss})
```
